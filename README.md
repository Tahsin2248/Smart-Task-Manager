# Smart Task Manager

A modern and responsive task management web application built with HTML, CSS, and JavaScript.

Smart Task Manager was developed as a personal portfolio project to strengthen front-end development skills and gain hands-on experience with real-world JavaScript application architecture. The project focuses on clean UI design, user experience, and practical task management features commonly found in productivity applications.

## Features

- Create, edit, and delete tasks
- Mark tasks as completed
- Restore completed tasks using undo functionality
- Real-time task progress tracking
- Dashboard statistics for total, completed, and pending tasks
- Search tasks instantly
- Filter tasks by status
- Sort tasks by priority, date, and creation time
- Drag and drop task reordering
- Import and export tasks using JSON files
- Dark mode and light mode support
- Toast notifications for user feedback
- Local storage support for persistent data
- Fully responsive design for desktop and mobile devices
